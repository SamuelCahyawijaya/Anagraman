# Anagraman

Anagram bot game

## Instruction

- Get all the dependencies
- `git clone`
- `git submodule update --init`
- `cd restclient-cpp && ./autogen.sh && ./configure && make install && cd ..`
- `make main`

## Dependencies

- libcurl `apt install libcurl4-openssl-dev`
- boost `apt install libboost-all-dev`
